Substring

We are going to get some letters from a string.


var anyString = 'CoderDojo';

console.log(anyString.substring(0, 3));
// 'Cod'

console.log(anyString.substring(5, 7));
// 'Do'

Substring Method

 
str.substring(indexA, indexB)

substring() extracts characters from indexA up to but not including indexB.

length property

 
str.length

length is a property of string that returns the amount of characters in the string.


var str = 'CoderDojo';

console.log(str + ' is ' + str.length + ' long');
// 'CoderDojo is 9 long'
