Before you start

Make sure you have a decent text editor to work with. We advice using brackets or atom, both are free and opensource editors for web development.

All exercises are always ordered from easy to hard. All you need to do to get started is create a file index.html with the contents on the right. Put your javascript code between the script tags and double click the file to open it in your browser.
