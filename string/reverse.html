Javascript has a reverse method for array. This method will reverse the array.

Join Method

 
str = arr.join(separator)

Javascript has a join method for array. This method returns a string with all elements in the array behind eachother separated by the given separator.

var a = ['Wind', 'Rain', 'Fire'];

console.log(a.join(', '));
// 'Wind, Rain, Fire'
