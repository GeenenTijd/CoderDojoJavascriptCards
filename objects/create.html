}

    thisIsTheNameOfMyProperty: 'thisIsTheValue',
    cool: true

obj2.name
// CoderDojo

Object

 
var obj = {name: 'CoderDojo'}

An object is a list of properties, these properties have a name (name) and a value (CoderDojo). You can access a property by typing a dot. To get the name of obj we can type: obj.name

Writing some code

Now it's up to you to create some objects. To see what you have created you can log the objects.

console.log(obj) logs obj in your console. You can open your console in your browser with F12 or CTRL+SHIFT+J.
